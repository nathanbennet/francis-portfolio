Know your files:
--------------------------------------------------------------------------------------
- main.py 
is where the app runs

--------------------------------------------------------------------------------------
- requirements.txt
are the dependency the app uses

--------------------------------------------------------------------------------------
- .gitignore
is to ignore some files to not put them on github

--------------------------------------------------------------------------------------
- templates/ 
all the HTML files there (the pages)
to edit a link in a html file, just find the link inside the a tag:

    ex: <a href="YOUR-LINK-HERE" > NAME_OF_THE_LINK </a>
    <a href="https://www.google.com/"> Google </a>

--------------------------------------------------------------------------------------
- static/assets/ 
the css style file, javascript files and images

--------------------------------------------------------------------------------------
- data/csv 
here is a csv file called emails.csv, this is where your 
form data will be stored when someone submit it in the contact form. 
So you can open the file sometimes to see if you get some messages. :)

--------------------------------------------------------------------------------------
